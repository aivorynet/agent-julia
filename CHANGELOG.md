# Changelog

All notable changes to the AIVory Monitor Julia Agent will be documented in this file.

This project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-02-16

### Added
- Exception capture via try/catch patterns
- Base.atexit hook for shutdown reporting
- WebSocket transport via HTTP.jl
- Configurable sampling rate and capture depth
- Environment variable configuration
- Julia 1.6+ support
